    <div class="post-content"><p>I recently made these <a href="http://www.marthastewart.com/portal/site/mslo/menuitem.0e0eb51a2e6b5ad593598e10d373a0a0/?vgnextoid=f229349d52e38110VgnVCM1000003d370a0aRCRD&amp;autonomy_kw=textured%20bracelets&amp;rsc=header_1" target="_blank">textured string bracelets</a> from April&#8217;s issue of <a href="http://www.marthastewart.com/portal/site/mslo/menuitem.4af27a8e9e64e1611e3bf410b5900aa0/?vgnextoid=36c9cf380e1dd010VgnVCM1000005b09a00aRCRD&amp;vgnextfmt=default" target="_blank">Martha Stewart Living</a> as a birthday gift for my sister. While a little time consuming, the final product is really great. I like that each time you can create a new color scheme and design. One thing I noticed however, is that the instructions said that when you&#8217;re done with a color, &#8220;Cut the string, leaving a 3-inch tail; thread through a needle. Thread the tail under strings on inner side of bangle to secure.&#8221; I found that this was not a very solid finishing step (or simple), because the tail end could easily get loose or unravel. So instead, each time I was finished with a color, I would glue both the end and the beginning of the string to the bangle and start wrapping the new color around the two strands. This made me feel a lot better about how it would hold up in every day wear. It also provides a more seamless look if you are using multiple colors.</p>
<p><img src="http://evie-s.com/news/wp-content/uploads/2008/03/bangles.jpg" alt="Textured Bangles" /></p>
<p>I&#8217;m sure it&#8217;s not the only time someone&#8217;s improved on a beautiful Martha idea! It just goes to show you that Martha isn&#8217;t perfect after all. I&#8217;m a big fan, so she&#8217;s still wonderful to me.</p>
